# FreCLI
this is a small tool created for NeoArc engine
its a box drawing tool currently in version v[0.0.1] (see include/fc/frecli.h)


if you want to use frecli download the file libfrecli.a from the lib/ folder put it a /lib folder in your new project and use `-L./lib -lfrecli` or if you want a other folder then rename `-L./lib` to smth other.
